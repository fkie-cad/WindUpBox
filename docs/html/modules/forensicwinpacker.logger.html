class windupbox.logger.logger.ConsoleHandlerFormatter

Bases: Formatter

custom formatter that is used for logging to console

format(record)

Format the specified record as text.

The record’s attribute dictionary is used as the operand to a string formatting operation which yields the returned string. Before formatting the dictionary, a couple of preparatory steps are carried out. The message attribute of the record is computed using LogRecord.getMessage(). If the formatting string uses the time (as determined by a call to usesTime(), formatTime() is called to format the event time. If there is exception information, it is formatted using formatException() and appended to the message.

setup up logging for a project

Parameters

  • loglevel_console – contains the loglevel such that no log messages with lower loglevel are shown in the console

  • loglevel_file – contains the loglevel such that no log messages with lower loglevel are shown in the logfile

  • logfile – path of the logfile that should be used (None if no logfile should be created)

  • console – True if log should be printed to console and False if not
