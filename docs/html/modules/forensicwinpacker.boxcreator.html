windupbox.boxcreator.selector.select_boxcreator_by_windows_info(windows_info: WindowsInfo) → WindowsBoxCreator

finds the best matching boxcreator class by their box_creator_info attribute. therefore if two matches are found the more precise on are used. So for instance if the filter of a boxcreator class B1 includes only windows_version and version and a filter of a boxcreator class B2 includes additionally the edition and both filters match the boxcreator class B2 will be used.
