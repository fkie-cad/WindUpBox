************
Basic Usage
************

The WindUpBox tool enables users to create minimal and customizable windows vagrant boxes.
Before the creation of a box a user first need to choose which windows version, edition and so on should be used.
How to list and filter those can be found in :doc:`/basics/listavailablesystems/index`.
Afterwards the box creation und customization takes place.
An walk through on how to that can be found in :doc:`/basics/howtocreateawindowsbox/index`.
