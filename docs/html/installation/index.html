Installation

To install the windupbox clone the repository, navigate into the directory and run pip install .. To check whether the tool is successfully installed and ready to use run windupbox -v.

Requirements

To use the tools main functionality of creation vagrant boxes two programs are required. The first one is VirtualBox , which is used as a hypervisor in the box creation process. As a second tool packer will be used for automating the os installation process and creating the vagrant box. Before using the tool make sure that VirtualBox is installed and that packer is downloaded. Additionally it is needed to add the packer executable path to the PATH environment variable.
